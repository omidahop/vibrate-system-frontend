## نمای کلی

سیستم ویبره یک برنامه وب پیشرفته برای ثبت و تحلیل داده‌های ویبره تجهیزات صنعتی است.

### ویژگی‌های اصلی

- ثبت داده‌های ویبره برای تجهیزات مختلف
- نمودارهای تحلیلی قدرتمند
- سیستم هشدار هوشمند
- قابلیت کار آفلاین
- طراحی ریسپانسیو
